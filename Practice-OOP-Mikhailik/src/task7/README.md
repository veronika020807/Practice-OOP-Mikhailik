# Графічний інтерфейс 

## Вступ до **Swing**

**Swing** — це бібліотека для створення графічного інтерфейсу користувача (GUI) у Java. Вона є частиною `javax.swing` і дозволяє створювати вікна, кнопки, текстові поля, таблиці та інші елементи інтерфейсу.
